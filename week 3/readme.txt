Exploratory Data Analysis:
In this stage you will become more familiar with the data set and it will start to take shape. In this assignment you will:

Understand the structure of datasets and visualize distributions of key variables.

Identify, handle, and analyze missing data and outliers using statistical techniques such as IQR and standard deviation thresholds.

Compute and interpret statistical measures, including median, IQR, and bounds.

Analyze relationships between variables through cross-tabulations, scatter plots, and heatmaps.

Examine trends in job satisfaction, programming language preferences, remote work, and compensation.

Perform comprehensive exploratory data analysis (EDA) by integrating multiple visualization and analytical techniques.
